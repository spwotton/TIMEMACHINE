# Quick Start Guide

## Fastest Way to Run

### Option 1: Direct Browser (No Server)
```bash
# Just open the file directly
open index.html
# or
firefox index.html
# or
google-chrome index.html
```

### Option 2: Python Server
```bash
python3 -m http.server 8418
# Open: http://localhost:8418
```

### Option 3: Node.js Server
```bash
npx http-server -p 8418
# Open: http://localhost:8418
```

### Option 4: npm script
```bash
npm start
# Open: http://localhost:8418
```

## Troubleshooting

**No 3D scene?**
- Make sure WebGL is enabled in your browser
- Try Chrome, Firefox, or Edge (not IE)

**No audio?**
- Click on the page first
- Check if tab is muted
- Check browser audio permissions

**Slow performance?**
- Close other tabs
- The effects are GPU-intensive
- Try a smaller window size
